# Workshop: KI im Journalismus

> **[Workshop starten](https://felixboehm.github.io/open-learn/#/add?source=https://felixboehm.github.io/workshop-AI-im-Journalismus/index.yaml)**

Ein interaktiver Workshop zum Einsatz von KI-Tools in der journalistischen Praxis.

## Content

### German (5 Lessons)
1. **Grundlagen: KI im Journalismus** — Was ist KI und wie wird sie im Journalismus eingesetzt?
2. **KI-Tools für die Recherche** — Wie KI bei der journalistischen Recherche unterstützt
3. **Texterstellung mit KI** — Automatisierte Texte und KI-gestützte Schreibassistenten
4. **Faktencheck und Verifikation** — KI-gestützte Faktenprüfung und Deepfake-Erkennung
5. **Ethik und Verantwortung** — Ethische Leitlinien für den KI-Einsatz im Journalismus
